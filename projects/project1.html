Description

Built a command-line program that encrypts alphabetic text using the Hill cipher. The program reads a key matrix from a file (2x2 up to 9x9) and encrypts a plaintext file, producing a ciphertext output. Emphasis was placed on correct matrix handling, modular arithmetic, and clean file I/O.

Skills Learned

  • Matrix operations and modular arithmetic
  • File processing and input validation
  • Implementing classical cryptography algorithms
  • Debugging and testing edge cases
